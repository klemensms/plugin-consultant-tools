# Sample Skill

This is a placeholder skill to demonstrate the structure.

## How to Add Your Skills

1. Create a new folder under `skills/` with your skill name
2. Add a `SKILL.md` file with:
   - YAML frontmatter (name, description, triggers)
   - Markdown content with instructions

## Example Structure

```
skills/
  ado-helper/
    SKILL.md
  solution-export/
    SKILL.md
  plugin-dev/
    SKILL.md
```

Replace this file with your actual skill content.
